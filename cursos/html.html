Sobre o Curso

Este curso de HTML5 é perfeito para quem está começando no desenvolvimento web. Você aprenderá todos os conceitos fundamentais para estruturar páginas web de forma semântica e acessível.

O que você vai aprender

  • Estrutura básica de documentos HTML
  • Tags semânticas do HTML5
  • Formulários e validação
  • Incorporação de mídia (vídeo, áudio, iframes)
  • Acessibilidade na web
  • Boas práticas de SEO

Pré-requisitos

Nenhum conhecimento prévio é necessário. Este curso é projetado para iniciantes absolutos.
